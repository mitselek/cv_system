# Dokumenteerimisoskus / Documentation Skill

## Profiil

Oskan koostada selget, põhjalikku ja kasutajasõbralikku tehnilist dokumentatsiooni nii arendajatele kui ka lõppkasutajatele. Kasutan dokumenteerimisel parimaid praktikaid: struktuur, näited, selgitused, avalik läbipaistvus ja pidev ajakohastamine.

## Näide / Example

Avalik dokumentatsiooniprojekt: [ai-team/issues/68](https://github.com/mitselek/ai-team/issues/68)

- Kirjeldasin keeruka süsteemi töövoogu ja API kasutust avalikus GitHubi is­sues, mis võimaldas nii tiimiliikmetel kui ka välistel huvilistel kiiresti mõista lahenduse loogikat.
- Dokumentatsioon sisaldab: eesmärki, tehnilist tausta, kasutusnäiteid, API struktuuri, vastuste näiteid ja selgitusi.
- Sain positiivset tagasisidet tiimilt ja kasutan seda stiili ka teistes projektides.
