## et

### Väljakutse

Eesti kommunismiohvrite memoriaal Maarjamäel avati 2018. aastal ja vajati täpset andmetööd, et valida välja ja esitada hukkunud ohvrite nimed „Teekonna" mälestusseina nimetahvlitele. Andmete kvaliteet pidi olema kõrgeim, kuna tegemist on avaliku mälestusmärgiga.

### Töö ulatus

- **22 000+ nime** esialgsed nimetahvlid (memoriaal avati 23.08.2018)
- **Perioodilised lisatahvlid** (2018-2024): Uued hukkunud ohvrid lisati kord või kaks aastas, kui uurimistöö käigus andmed täpsustusid
- **Elektrooniline andmebaas:** Üle 100 000 kommunismiohvri andmed (hukkunud, vangistatud, küüditatud)

### Lahendus

Kogu andmetöö toimus minu loodud töövahendite ja päringute abil:

- **Google Sheets päringud:** Spetsiaalsed töökeskkonnad ajaloolastele andmete otsimiseks ja võrdlemiseks
- **MySQL andmebaasipäringud:** Andmete väljavalimine eMem andmebaasist
- **Kvaliteedikontroll:** Andmete valideerimine ja täpsustamine
- **Koostöö ajaloolastega:** Pidev tagasiside ja andmete korrigeerimine uurijate märkuste põhjal
- **Iteratiivne protsess:** Lisatahvlite andmete ettevalmistamine iga uue paigalduse jaoks

### Tulemus

- **Maarjamäe memoriaal:** Üks peamisi nähtavaid tulemusi Eesti Mälu Instituudi töös
- **22 000+ nime mälestusseinal:** Hukkunud kommunismiohvrite avalik mälestusmärk
- **Perioodilised lisatahvlid (2018-2024):** Pidev andmete täpsustamine ja uute ohvrite lisamine
- **Avalik mõju:** Riiklikult oluline mälestusmärk, mida külastavad tuhanded inimesed aastas
- **Andmete usaldusväärsus:** Kõrge kvaliteediga andmetetöö ajalooliselt tähtsa projekti jaoks

## en

### Challenge

The Estonian communist victims memorial at Maarjamäe opened in 2018 and required precise data work to select and present the names of deceased victims on the "Journey" memory wall name plates. Data quality had to be of the highest standard as this is a public memorial.

### Scope of Work

- **22,000+ names** on initial name plates (memorial opened 23.08.2018)
- **Periodic supplementary plates** (2018-2024): New deceased victims added once or twice a year as research clarified the data
- **Electronic database:** Over 100,000 communist victim records (deceased, imprisoned, deported)

### Solution

All data work was performed using tools and queries I created:

- **Google Sheets queries:** Special work environments for historians to search and compare data
- **MySQL database queries:** Data extraction from the eMem database
- **Quality control:** Data validation and refinement
- **Collaboration with historians:** Continuous feedback and data correction based on researchers' notes
- **Iterative process:** Preparing supplementary plate data for each new installation

### Result

- **Maarjamäe Memorial:** One of the main visible outputs of Estonian Memory Institute's work
- **22,000+ names on memory wall:** Public memorial for deceased communist victims
- **Periodic supplementary plates (2018-2024):** Continuous data refinement and addition of new victims
- **Public impact:** Nationally significant memorial visited by thousands annually
- **Data reliability:** High-quality data work for historically important project
