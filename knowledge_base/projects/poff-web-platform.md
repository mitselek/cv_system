# PÖFF Web Platform

**Complete web platform for Black Nights Film Festival (PÖFF) - one of Europe's largest film festivals**

Led technical architecture and development of the entire digital infrastructure for PÖFF (Pimedate Ööde Filmifestival), Estonia's premier international film festival. The platform serves multiple festival domains and handles the complete festival logistics system.

## Technology Stack

### Backend

- **Node.js** - Server runtime
- **Strapi CMS** - Headless CMS for content management
- **JavaScript** - Primary language (42.4% of codebase)
- **PostgreSQL** - Database

### Frontend

- **Pug** - Templating engine (36.3% of codebase)
- **Stylus** - CSS preprocessor (19.4% of codebase)
- **Static Site Generator (SSG)** - Performance optimization
- **JavaScript** - Interactive features

### Deployment & Operations

- **Shell scripts** - Deployment automation (1.8% of codebase)
- **Database backups** - Data integrity
- **AWS CloudFront** - CDN and hosting

## Platform Scope

### Festival Domains

- **poff.ee** - Main festival site
- **justfilm.ee** - Youth and children's film festival
- **kinoff.ee** - Russian film festival
- **industry.poff.ee** - Industry professional platform
- **shorts.poff.ee** - Short film festival

### Features

- Festival program management
- Film catalog and screening schedules
- Ticket sales integration
- Industry professional networking
- Multi-festival content management
- Logistics system integration
