# Runoviisid

**Musical score publishing using cartographic visualization technology**

"Runoviisid" means "song melodies" in Estonian. This project represents an innovative approach to musical score visualization and publishing, using CartoCSS (a cartographic styling language) to render musical notation.

## Technology

- **Primary Language:** CartoCSS
- **Purpose:** Musical score visualization and publishing
- **Approach:** Leverages mapping/cartographic technology for music notation

## Innovation

This project demonstrates creative technical problem-solving by repurposing geographic visualization tools (CartoCSS) for musical notation. The approach allows for:

- Precise control over musical symbol placement
- Scalable vector-based score rendering
- Web-compatible musical notation display
