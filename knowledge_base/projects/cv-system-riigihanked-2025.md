## en

Public GitHub project for automated analysis and fit assessment of Estonian public procurements.

**Features:**

- Python and Go harvesters for riigihanked.riik.ee API
- API endpoint documentation (API_DISCOVERY.md)
- Automated procurement notice download and structuring
- Procurement registry creation and management (REGISTRY.md)
- Requirement extraction from procurement documents
- Fit score calculation based on knowledge base

**Technical Approach:**

- REST API integration with public procurement system
- Document processing (DOCX, PDF)
- Structured data management
- Automated requirement comparison

**Project Goal:**

Gaining practical understanding of Estonian public procurement system, document structures, and requirements through a technical approach.

**Repository:** https://github.com/mitselek/cv_system (subfolder: `riigihanked/`)
