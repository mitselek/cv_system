# CV System

**Comprehensive career management and application tracking system**

An integrated system for managing career documentation, procurement applications, and knowledge base organization. This meta-project represents the systematic approach to career management, combining documentation, automation, and knowledge organization.

## Purpose

### Career Management

- Structured knowledge base of experiences, skills, achievements
- Procurement opportunity tracking and analysis
- Application document generation
- Portfolio evidence organization

### Documentation System

- **Markdown-based** knowledge base
- **Obsidian-compatible** wiki-link structure
- **Git version control** for history tracking
- **Automated workflows** for document generation

## Structure

### Knowledge Base (`knowledge_base/`)

- **experiences/** - Employment history with detailed context
- **skills/** - Technical and professional competencies
- **achievements/** - Specific accomplishments with evidence
- **certifications/** - Professional certifications
- **education/** - Academic background
- **projects/** - GitHub project documentation
- **languages/** - Language proficiency

### Procurement Management (`riigihanked/`)

- Procurement opportunity tracking
- Document harvesting and analysis
- Application preparation workflows
- Decision documentation

### Automation (`scripts/`)

- Context compilation
- Document conversion (Markdown → PDF)
- Estonian grammar correction
- Automated workflows

### Knowledge Management
