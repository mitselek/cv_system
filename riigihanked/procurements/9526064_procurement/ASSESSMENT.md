## Procurement Structure

### Contract Object

**Primary Goal:** Replace Scoro CRM system currently used at Kultuurikatel and Salme Kultuurikeskus with modern all-in-one CRM solution supporting:

1. **Customer Relationship Management (CRM):**

   - 360-degree customer view (contacts, communication history, projects, invoices)
   - Integrated with projects and finances

2. **Sales Pipeline:**

   - Customizable deal stages, activities, forecasts
   - Sales team and activity reporting

3. **Integrated Calendar and Resource Planning:**

   - Manage: rooms, people, tasks, invoices, quotes, rental orders, projects, time tracking
   - Link calendar events to contacts, quotes, tasks, invoices, projects
   - Sync with Google Calendar and Outlook

4. **Quote Management:**

   - Create quotes
   - Convert quotes to contracts and invoices

5. **Invoice Management and Billing:**

   - Create invoices from logged time or quotes
   - Compatible with Finbite e-invoicing

6. **Time Tracking:**

   - Track project and activity hours
   - Integration with invoicing

7. **Project Management:**
   - Tasks, calendars, resources, time tracking
   - Covers full project lifecycle (quotes → planning → execution → billing)
   - Gantt charts, resource planning, real-time progress tracking

### Multi-Unit Requirements

System must support 3 sub-units with separate workflows but unified overview:

- **Tallinna Kultuurikatel** (currently using Scoro)
- **Mere kultuurikeskus** (currently no CRM)
- **Salme kultuurikeskus** (currently using Scoro)

**Key capabilities:**

- Separate data and workflow management per sub-unit
- Unified overview across all event centers
- Same system for different users with role-based permissions
- Calendar view for events and resources across all units
- Reporting at sub-unit and institution-wide level
