## Project Scope

RIK's E-catalog is a post-procurement contract management platform integrated with the National Procurement Register (RHR). Contractors and procurers use it to manage orders, products/services, and fulfillment for framework agreements established through RHR e-catalog procurements.

**Core System:**

- **Backend:** Java platform, Spring Boot framework
- **Frontend:** Angular
- **Database:** PostgreSQL 10
- **Search:** Elasticsearch (performance issues - optimization needed)
- **Orchestration:** Requires integration with RHR user management
- **Integration:** X-Road and REST APIs

**Expected Work Types:**

- Performance optimization (SLA compliance critical)
- Elastic Search tuning (explicit priority)
- User interface improvements
- Role/permissions system refactoring
- New feature development
- System analysis and documentation
- Testing (including automated tests)
- Occasional training

**Working Model:**

- Hourly-rate framework agreement (bid on €/hour)
- Work ordered via specific assignments with scope + deadline
- Contractor proposes effort estimate and schedule
- Delivery in phases with acceptance protocols
- JIRA-based task management
- Confluence documentation
