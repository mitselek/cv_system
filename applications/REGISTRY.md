# Job Applications Registry

This file tracks all job applications generated from the CV system. Each application should be registered here with comprehensive tracking information.

## Status Definitions

- **Draft:** Application materials being prepared, not yet complete
- **Ready:** Application package complete, ready to submit
- **Submitted:** Application sent to employer
- **Interview:** Interview scheduled or completed
- **Offer:** Job offer received
- **Accepted:** Offer accepted, position secured
- **Rejected:** Application rejected by employer
- **Withdrawn:** Application withdrawn by candidate
