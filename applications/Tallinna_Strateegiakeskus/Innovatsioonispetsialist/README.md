## Kandideerimise strateegia

**Lähenemisviis: Aus karjääripöörde narratiiv**

Kandideerimine põhineb **ülekantavate oskuste** tugevusel, mitte otsese kogemuse vastavusel:

1. **Rõhutada ülekantavaid oskusi:**
   - Keerukate süsteemide juurutamine
   - Mitme huvigrupi koordineerimine
   - Kasutajate edukas kaasamine

2. **Tunnistada lünka:**
   - Karjääriüleminek innovatsiooni-ökosüsteemi töösse
   - Soov tuua süsteemset mõtlemist avalikku sektorisse

3. **Positsioneerida PÖFF kogemus:**
   - Lähim analoog: kultuuriinnovatsioon, rahvusvaheline koostöö, keeruline logistika
   - Mitme huvig

rupi kaasamine suurürituse kontekstis

4. **Motivatsioon:**
   - Panus süsteemse mõtlemise viimine omavalitsuse innovatsiooni koordineerimisse
   - Tehnilise rakenduskogemuse kasutamine piloteerimisprojektide toetamisel

**OLULINE:** See EI OLE otsene kogemuse vastavus. See on tugev profiil inimesele, kes on valmis tooma värskeid perspektiive tehnilisest rakendamisest innovatsiooni koordineerimisse.
