# Full-stack arendaja

## Töö kirjeldus

Sinu peamised ülesanded:

- moodsa ja kasutajasõbraliku UI loomine (Svelte/SvelteKit);
- backend arendus Node.js (Express, NestJS) ja Java (Spring Boot);
- andmebaaside haldus (PostgreSQL, Oracle, Cloudflare D1);
- testimine (Cypress, Playwright, Testing Library), DevOps, pilveteenused (Docker, Cloudflare Workers);
- koostöö meeskonna ja partneritega.

## Ootused kandidaadile

Ootused Sulle:

- väga hea Svelte/SvelteKit oskus ning kogemus UI ja jõudluse optimeerimisel;
- tugev Node.js ja Java tundmine, REST API-de loomise kogemus;
- väga head oskused PostgreSQL ja Oracle (PLSQL) kasutamisel;
- kogemus pilveteenuste ja konteineritega;
- hea arusaam testimisest ja Git-põhistest töövoogudest;
- hea eesti ja inglise keele oskus;
- iseseisvus, süsteemsus ja koostöötahe.

## Ettevõte pakub

Meie juures ootab Sind võimalus panustada Eesti Raudtee digipöördesse, arendades nullist uut kaasaegset raudtee tuumik IT-lahendust. Meil on kokkuhoidev meeskond ja kaasamõtlevad juhid, kes väärtustavad innovatsiooni, avatust ja tõhusust. Töökoht asub Telliskivi TLN-s, kus on tasuta parkimisvõimalus. Töötajad saavad osa mitmetest hüvedest, mis toetavad nende tervist ja heaolu.
